📋 GEC-2015 Methodology

Stage of Extraction (SoE) = (Total Extraction / Extractable Resources) × 100

The Ground Water Resource Estimation Committee (GEC-2015) methodology is used for assessing groundwater resources in India. Categories are assigned based on the Stage of Extraction percentage.

For bulk data import, use the API endpoint: POST /api/data/import

The import endpoint accepts JSON with arrays for states, districts, and groundwater data.
